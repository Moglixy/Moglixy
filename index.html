Part 1: Get Your Google Client ID

  1. Go to the Google Cloud Console and create a new project.
  2. In the sidebar, navigate to APIs & Services > Library.
  3. Search for and enable the Gmail API, the Google Sheets API, and the Google Picker API.
  4. Go to APIs & Services > Credentials. Click + CREATE CREDENTIALS and select OAuth client ID.
  5. Configure the OAuth consent screen if prompted (select User Type: External, then provide an app name and your email).
  6. For Application type, select Web application.
  7. Under Authorized redirect URIs, click + ADD URI and enter the exact URL of this page. You can copy it from your browser's address bar.
  8. Click Create, and copy your Client ID.

Part 2: Get Your Gemini API Key

  1. Go to Google AI Studio.
  2. Sign in with your Google account.
  3. Click Create API key in new project.
  4. Copy your generated API key. This will also be used as your Developer Key for the Picker.
